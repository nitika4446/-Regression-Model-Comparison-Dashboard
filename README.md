# 📊 Regression Model Comparison Dashboard

A Streamlit-based Machine Learning project that compares multiple regression algorithms on the Mall Customers dataset. This project helps understand how different models perform on the same data and highlights the importance of choosing the right model.

---

## 🚀 Project Overview

Regression models are widely used in Machine Learning for predicting continuous values. However, each model behaves differently depending on the dataset.

This project provides:

* 📈 Performance comparison of multiple regression models
* 📊 Visualization of results
* ⚙️ Real-world dataset implementation
* 🌐 Interactive dashboard using Streamlit

---

## 🧠 Models Implemented

* Linear Regression
* Stochastic Gradient Descent (SGD)
* Decision Tree Regression
* Random Forest Regression
* K-Nearest Neighbors (KNN)
* Support Vector Regression (SVR) *(optional)*
* Neural Network (MLPRegressor) *(optional)*
* XGBoost Regression *(optional)*
* Polynomial Regression

---

## 📂 Dataset

**Mall_Customers.csv**

Features used:

* Age
* Annual Income (k$)
* Gender

Target:

* Spending Score (1–100)

## 🖥️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/regression-model-comparison.git
cd regression-model-comparison
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the App

```bash
streamlit run app.py
```

---

## 📸 Output Preview

* Model performance table
* Horizontal bar chart comparing R² scores

---

## 🔥 Key Learnings

* No single model works best for all datasets
* Ensemble methods like Random Forest often perform better
* Scaling is crucial for models like KNN, SVR, and Neural Networks
* Model selection depends on data complexity and use case
